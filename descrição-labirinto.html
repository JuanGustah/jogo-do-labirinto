Como Jogar

Bem-vindo ao tutorial do Labirinto! Aqui você aprenderá como jogar o nosso jogo.

Objetivo do Jogo

O objetivo do jogo é guiar o seu personagem pelo labirinto até a célula verde que representa o final do labirinto.

Controles

  • Use as setas do teclado para mover seu personagem:
    • ↑ Seta para cima: move para cima
    • ↓ Seta para baixo: move para baixo
    • ← Seta para esquerda: move para esquerda
    • → Seta para direita: move para direita

Velhinho

O velhinho se move pelo labirinto em direção ao seu personagem. Evite ser pego por ele!

Modo Infinito

Se o mapa no modo infinito for impossivel de completar aperte "Command" + "R" para criar um novo mapa.

Você Está Pronto?

Agora que você conhece os controles e o objetivo, está pronto para jogar?! Se sim, boa sorte!
